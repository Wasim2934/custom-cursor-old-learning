#  Custom Cursor Animation — mix-blend-mode: difference

> ⚠️ **Archived Notice**  
> This project was built during my initial learning phase.  
> This repository is archived and kept only for history and progress tracking.  
> A better and more professional version of this project will be built in a new repository.

A clean and modern UI mini project where the default mouse cursor is replaced with a smooth,
animated circle that follows the pointer and blends uniquely with the background using  
`mix-blend-mode: difference`, creating a glowing inverse-color effect.

##  Live Demo - https://wasim2934.github.io/custom-cursor/

## Features

- Custom circular cursor  
- Smooth follow animation  
- `mix-blend-mode: difference` effect  
- Removes default system cursor  
- Lightweight — no libraries  
- DOM project

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (DOM)**
